# SkyNet Worldwide Express — Demo Context, Setup & Scenarios

## Context

SkyNet Worldwide Express UAE is the arm of a global independent courier network operating express delivery, courier and freight services across the Middle East, North Africa and the Indian Subcontinent, with its Dubai office serving as the regional hub. The network runs over 1000 offices worldwide.

SkyNet moves documents and parcels both locally within the UAE and internationally, serving corporate accounts (including e-commerce shippers) alongside individual customers. It competes directly with DPD, Aramex, DHL and FedEx on price, speed and service coverage in the UAE and wider GCC market.

During the meeting, the client asked to see how Salesforce can support a courier company's day-to-day operations — selling to corporate accounts, handling delivery complaints, and running the contact centre — following a DPD-style process, but in a light, stripped-down version.

Cost is the deciding factor. The client explicitly said they'd heard Salesforce is expensive, and want a very cost-effective, phased approach: a lean rollout in the UAE first, then expansion to GCC. The demo needs to visibly carry that message — a light configuration with minimal customisation, quick to launch, with a foundation that scales rather than a feature-heavy package.

Technical base: VRP already has two ready demos that should be merged rather than rebuilt from scratch. We are using the existing org where the demo for OceanAir is already set up (Twilio Flex + WhatsApp + Service Cloud Voice). Telephony and messaging are already working. The task is to add data for SkyNet and adapt the scenario. Data for OceanAir should not be deleted.
